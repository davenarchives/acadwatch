AcadWatch

AcadWatch predicts student dropout risk from academic performance, tuition status, demographics, socioeconomic indicators, and enrollment records. The project trains and evaluates multiple classifiers, then uses a tuned Random Forest model to support early intervention decisions.

Dataset Info

Student Dropout Dataset

Source: Kaggle student dropout dataset. This project uses 4,424 student records with 29 input features and one target column to classify Dropout versus Non-Dropout outcomes.

Target Variable

Binary target: Dropout (1) vs Non-Dropout (0).

Feature Families

Inputs include academic performance, financial status, family background, course enrollment, and demographic context.
